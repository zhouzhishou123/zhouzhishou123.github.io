三、数据类型之间转换的规则

  1. 其他数据类型转换成数字

    • parseInt/parseFloat的转换规则:

      处理的值是字符串，从字符串的左侧开始查找有效数字（遇到非有效数字字符则停止查找，如果第一个是非有效数字则为NaN）如果处理的值不是字符串需要先转换成字符串然后以上的规则进行处理。parseFloat能够识别小数点。

    • Number的转换规则（隐式转换一般都是基于Number做转换的）

      true –> 1
      false –> 0
      null –> 0
      “” –> 0
      undefined –> NaN
      字符串类型数据只有都是有效数字才会转换成数字否则结果都是NaN

    • 对象转换成数字

      先转换成字符串在转换成数字

  2. 其他数据类型转换成布尔值

    0 NaN null undefined 空字符串转换成布尔值为false,其他的情况为true

    附上大厂面试题

    1. parseInt("") // NaN
2. Number("")  // 0
3. isNaN("") // false
4. parseInt(null) // NaN
5. Number(null) // 0
6. isNaN(null) // false
7. parseInt("12px") // 12 
8. Number("12px") // NaN
9. isNaN("12px") // true
10. parseFloat("1.6px")+parseInt("1.2px")+typeof parseInt(null); // "2.6Number" 
11. isNaN(Number(!!Number(parseInt("0.8")))); // false
12. typeof !parseInt(null) + !isNaN(null); // "booleantrue"
